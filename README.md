# Features

- [x] Configuration via JSON-File
- [x] Welcome message(for server & user)
- [x] Clear channel
- [x] Roles by reaction
- [ ] [Server stats](https://files.dulliag.de/share/Discord_AL5lriRcmD.png)
- [x] Kick {Member} {Reason}
- [x] Ban {Member} {Reason}
- [x] Read an RSS-Feed
  - Done by Discord.RSS
- [x] Get stock informations(for specific stocks)
